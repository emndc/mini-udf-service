📋 SEÇENEK 1: RENDER.COM (HIZLI & BEDAVA) ✅ TAVSIYE
═══════════════════════════════════════════════════════════════════════════

ADIM 1: Hazırlık
─────────────────
cd production

# Git repository'e ekle (henüz yoksa)
git init
git add .
git commit -m "Add production mini_udf_service"

# GitHub'a push (henüz yapmadıysan)
git remote add origin https://github.com/YOUR_USERNAME/flask-udf-converter
git branch -M main
git push -u origin main

ADIM 2: Render.com Dashboard
─────────────────────────────
1. https://dashboard.render.com adresine git
2. "New +" → "Web Service"
3. GitHub repository'i seç (flask-udf-converter)
4. Ayarları yap:
   
   Name: mini-udf-service
   Environment: Python 3
   Region: Frankfurt (Avrupaya yakın) veya Singapore
   Branch: main
   Build Command: pip install -r production/requirements-prod.txt
   Start Command: cd production && gunicorn -w 4 --threads 2 -b 0.0.0.0:10000 mini_udf_service_secure:app

5. Environment Variables ekle:
   
   Key → Value
   ─────────────────────────────────────────
   FLASK_ENV → production
   ENVIRONMENT → production
   API_SECRET_KEY → sk_live_... (kendini oluştur)
   CORS_ORIGINS → https://yourapp.render.com
   PORT → 10000
   DEBUG → False

ADIM 3: Deploy
──────────────
"Create Web Service" butonuna tıkla → Otomatik deploy başlıyor!

URL: https://mini-udf-service.onrender.com (otomatik atanır)

ADIM 4: Test Et
────────────────
curl https://mini-udf-service.onrender.com/health

curl -X POST \
  -H "X-API-Key: sk_live_..." \
  -F "file=@sample.udf" \
  https://mini-udf-service.onrender.com/api/parse-udf

📋 SEÇENEK 4: DIGITAL OCEAN ($5/ay VPS)
═══════════════════════════════════════════════════════════════════════════

ADIM 1: Droplet Oluştur
────────────────────────
1. https://cloud.digitalocean.com login
2. "Create" → "Droplets"
3. "Ubuntu 22.04 LTS" seç
4. $5/month basic seç
5. Region: Frankfurt (Avrupa)
6. SSH key ekle (güvenlik için)
7. Create Droplet

ADIM 2: SSH ile Bağlan
──────────────────────
ssh root@YOUR_DROPLET_IP

ADIM 3: System Setup
─────────────────────
apt update && apt upgrade -y
apt install -y python3-pip python3-venv git nginx

ADIM 4: App Deploy Et
──────────────────────
cd /opt
git clone https://github.com/YOUR_USERNAME/flask-udf-converter
cd flask-udf-converter/production

# Virtual env
python3 -m venv venv
source venv/bin/activate
pip install -r requirements-prod.txt

ADIM 5: .env Dosyasını Oluştur
────────────────────────────────
nano .env

# CTRL+X, Y, ENTER (kaydet)

Şu yapıya yapıştır:
─────────────────
FLASK_ENV=production
ENVIRONMENT=production
API_SECRET_KEY=sk_live_... (oluştur!)
CORS_ORIGINS=https://yourdomain.com
HOST=127.0.0.1
PORT=5055

ADIM 6: Systemd Service Oluştur
─────────────────────────────────
sudo nano /etc/systemd/system/mini-udf.service

[Unit]
Description=Mini UDF Service
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/flask-udf-converter/production
ExecStart=/opt/flask-udf-converter/production/venv/bin/gunicorn -w 4 --threads 2 -b 127.0.0.1:5055 mini_udf_service_secure:app
Restart=on-failure
RestartSec=5s

[Install]
WantedBy=multi-user.target

# CTRL+X, Y (kaydet)

ADIM 7: Service Başlat
───────────────────────
sudo systemctl daemon-reload
sudo systemctl enable mini-udf.service
sudo systemctl start mini-udf.service
sudo systemctl status mini-udf.service

ADIM 8: Nginx Setup (Reverse Proxy)
─────────────────────────────────────
sudo nano /etc/nginx/sites-available/mini-udf

# nginx.conf dosyasynin içeriğini kopyala
# YOUR_DOMAIN_HERE yerine kendi domain'i yaz

sudo ln -s /etc/nginx/sites-available/mini-udf /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

ADIM 9: SSL Sertifikası (Let's Encrypt)
─────────────────────────────────────────
sudo apt install certbot python3-certbot-nginx
sudo certbot certonly --nginx -d yourdomain.com
# Email gir, terms accept et

URL: https://yourdomain.com
